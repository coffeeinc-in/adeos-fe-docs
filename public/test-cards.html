Nextra Cards Padding Test

Before CSS Overrides (Basic styling)

Low Padding

This card has minimal padding. Text appears very close to the borders, which can look cramped and hard to read.

Accessibility Issue

Poor spacing makes it difficult to focus on individual cards and can impact readability significantly.

User Experience

Cards with insufficient padding create a cluttered appearance and poor visual hierarchy.

After CSS Overrides (Improved)

Proper Padding

This card has appropriate padding (1.5rem) that provides breathing room and better visual separation from the borders.

Better Readability

Generous padding improves text readability and makes the content easier to scan and focus on.

Enhanced UX

Well-spaced cards create a more professional appearance and better visual hierarchy for documentation.
